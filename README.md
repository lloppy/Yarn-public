# Yarn — Делитесь личным дневником

**Yarn** — мобильное приложение для ведения приватного совместного дневника с близкими.

Соцсети ориентированы на публичность. Личные дневники — только для одного. Облачные альбомы превращаются в свалку файлов. Ценные воспоминания теряются, а у пар и друзей нет совместной истории. Yarn решает эту проблему — вы ведёте личную хронику событий и сами выбираете, какими записями и с кем поделиться.

---

## Как это работает

### 1. «Давай куда-нибудь сходим?»

Всё начинается с идеи — сходить в кафе, съездить в путешествие или просто погулять. Yarn превращает эту идею в совместное приключение.

---

<p align="center">

Откройте Yarn и создайте событие. Отметьте несколько мест, куда можно пойти, и пригласите друзей.

Во время события публикуем истории — сторис прямо из поездки. Лента обновляется в реальном времени.
